Search Feature in Chrome (OmniBus Search)

      No Manual Steps.. This automatically adds Bazinga to the list of search engines, so that you can directly start searching from addressbar.. Just start typing ppmtech and it asks you to press tab to search in bazinga ..    Piece of cake..

Search Feature in Firefox

       Well, in Firefox 'Address bar' & 'Search bar' are different so we will get a '+' sign on the 'Search Bar' (Pic 1 below). And when we press it we will see option to add it to the list of search items.. Not as good as chrome but still ok..
